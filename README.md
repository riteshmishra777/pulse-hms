🏥 Pulse HMS – Hospital Management System

Pulse HMS is a modern full-stack web application designed to digitize and optimize hospital workflows. It provides secure, role-based access for admins, doctors, and patients, ensuring efficient management of appointments, medical records, and billing in a centralized system.

🚀 Features
👤 User Authentication & Authorization
Secure login/register system
Role-based access (Admin, Doctor, Patient)
🩺 Doctor Management
Manage doctor profiles
View schedules and appointments
🧑‍🤝‍🧑 Patient Management
Register and manage patient details
Maintain medical records
📅 Appointment System
Book, update, and cancel appointments
Real-time scheduling
💳 Billing System
Generate and manage invoices
Track payments
📊 Dashboard
Separate dashboards for each role
Overview of hospital activities

🛠️ Tech Stack
Frontend:
React (with TypeScript)
Vite (for fast build & development)

Backend:
Java Spring Boot (REST APIs)

Database:
MySQL
